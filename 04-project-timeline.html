About this project

We are building an AI-enabled UX writing capability that helps product design teams create and assess on-brand copy at scale, while protecting quality through clear standards and governance. The agent does not replace writers. It applies their standards, handles volume, and flags work that needs human review.

Platform: Copilot Studio (confirmed) · KB: 10–12 SharePoint MD files · T1 channels: App, Nova, ATM · Access: Teams · Figma plugin T2 · Key requirement: Project context memory — agent remembers project brief across sessions. Phase 1 exit condition: All KB files complete, agent configured, Layer 0 signed off by legal, basic test passing.
